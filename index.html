Segurança Transparente

VacLook é um sistema de criptografia projetado para garantir a máxima segurança e privacidade no armazenamento de dados. Nascido da necessidade de guardar informações sensíveis de forma interessante e impenetrável, o sistema permite que você proteja textos, imagens e arquivos diversos em um ambiente client-side, longe dos olhos da internet. Entenda como seus dados são protegidos.

AES-256-GCM
Utilizamos o padrão de criptografia avançada (AES) com chaves de 256 bits em modo GCM (Galois/Counter Mode). Isso garante não apenas a confidencialidade dos seus dados, mas também a integridade, impedindo que sejam alterados sem a chave correta.
PBKDF2 & SHA-512
Sua senha nunca é usada diretamente. Usamos PBKDF2 com 600.000 iterações e SHA-512 para derivar uma chave criptográfica forte, tornando ataques de força bruta extremamente difíceis e lentos.
Client-Side
Tudo acontece no seu navegador. Seus dados nunca são enviados para nenhum servidor. A criptografia é executada localmente usando a API Web Crypto do seu dispositivo com máxima segurança.
Open Source
Este sistema é de código aberto e auditável no GitHub. Criado por oVacilo e usado na comunidade thevacilos, permite a verificação pública do código-fonte, reduzindo riscos de backdoors, vulnerabilidades ou qualquer implementação maliciosa.
